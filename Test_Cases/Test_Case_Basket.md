#Test Case: Basket Page

**Test Case ID:** TC_UA_Basket_002
**Title:** Add product to basket from product page

## Preconditions
- User is on the product page
- Product is available for purchase
- User is not required to be logged in

## Test Steps
1. Open the website
2. Navigate to a product page
3. Click the "Add to Cart" button
4. Open the Basket/Cart page

## Expected Result
- Product is added to the basket
- Correct product name is displayed
- Correct product price is displayed
- Product quantity is set to 1
